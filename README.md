# Learning Git Basics

Учебный репозиторий для изучения базовых команд Git и работы с GitHub.

## Цель проекта
Закрепить на практике:
- создание репозитория
- работу с файлами
- коммиты
- ветки (branches)
- базовую навигацию в GitHub

## Что здесь есть
- текстовые файлы для тренировки коммитов
- примеры работы с ветками
- история изменений (commit history)

## Статус проекта
Учебный проект (в процессе обучения)
